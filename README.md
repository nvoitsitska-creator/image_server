# Image Server — тестовий проєкт

Навчальний проєкт для практики роботи з Docker, Python HTTP-сервером та Nginx.

Проєкт містить **навмисні помилки**, які потрібно знайти та виправити.

## Завдання

### 1. Виправити помилки (мінімум 2)

- **Кнопка "Copy"** — копіювання URL завантаженого зображення не працює
- **Видалення зображень** — при видаленні зображення воно зникає лише з localStorage браузера, але файл залишається в локальній папці `images/` на сервері

### 2. Рефакторинг

Проаналізуйте код (`app.py`, JS-файли у `static/`) та проведіть рефакторинг:
покращте структуру, читабельність, усуньте дублювання коду.

## Запуск

```bash
docker compose up --build
```

Відкрийте http://localhost:8080 у браузері.

## Технології

- Python (http.server, Pillow)
- Nginx
- Docker / Docker Compose
- HTML / CSS / JavaScript
